# online-shop-frontend

## Build Setup

```bash
# install dependencies
$ npm install

# main nuxt.config.js
```
axios: {
    baseURL: '' 
  }
  
  env: {
      PAYPAL_CLIENT_ID: ""
    }
```

# admin nuxt.config.js
```
axios: {
    baseURL: '' 
  }
  
  env: {
      PUSHER_APP_KEY: ""
    }
```

# serve with hot reload at localhost:3000 for website and localhost:4000 for admin
$ npm run dev

# build for production and launch server
$ npm run build
